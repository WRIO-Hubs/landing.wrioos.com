Linked Data

Semantics · Magic sauce

Information rules the world, in which data is its currency. World of Big Data doubles every 18 months and automated processing of data is a must, but less than 5% of pages are semantic.
Structured data (Linked Data) is a standardized key:value format (JSON-LD) that classifying the page content; i.e., on a recipe page, what are the ingredients, the cooking time and temperature, the calories, and so on. It builds upon standard Web technologies such as HTTP, RDF and URIs, but rather than using them to serve web pages for human readers, it extends them to share information in a way that can be read automatically by computers. Based on schema.org, supported by all major search engines.

Blockchain stores assets and metadata. Metadata is data [information] that provides information about other data.

What WRIO Internet OS is

User-centric browsing experience

The WRIO Internet OS platform greatly improves the browsing experience of the everyday Internet user. By utilizing blockchain data storage and it’s ability to analyze and output machine-readable data, the user experience is exponentially improved compared to the current norm.

Correlated to the user’s interests stored in the profile. If there is a match, the page is displayed in the user’s web dashboard, which is a key element of user-centric web.


Single sign on

Single sign on through the blockchain ledger makes it easier than ever for users to access all their data from one sign on point rather than multiple email, password logins.

Spam-proof

Security
