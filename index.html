WRIO Internet OS

All you need is a browser

An easy-to-use, privacy respectful and browser-based open-source operating system with blockchain-powered cloud apps enabled in a click that use public decentralized storage of machine-readable Linked Data. This allows automatic data processing of all information uploaded into the web based on your interests.

Now, this is what WRIO Internet OS is ­— an upgrade to a user-centric Web 3.0 with a predictive services and search engines of new generation.

Features

User-centric browsing experience · Web 3.0

The WRIO Internet OS platform greatly improves the browsing experience of the everyday internet user. By utilizing blockchain data storage and it’s ability to analyze and output machine readable data, the user experience is exponentially improved compared to the current norm.


  • grade Single sign on through the blockchain ledger makes it easier than ever for users to access all their data from one sign on point rather than multiple email, password logins
  • grade Spam and irrelevant advertisements are virtually eliminated through the Credit and reputation system enabling a more user friendly browsing environment
  • grade Security is unmatched through the use of blockchain and single sign on
  • grade Multiple page themes to choose from and automatic data processing makes every page on the new web individual to the user, again, promoting the new user centric browsing experience
